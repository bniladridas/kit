## kit auth login

Authenticate with a provider

### Synopsis

Store an authentication token for a service provider.

```
kit auth login [flags]
```

### Options

```
      --client-id string   OAuth client ID (defaults to GitHub CLI client ID)
  -h, --help               help for login
  -p, --provider string    provider: github (default "github")
  -t, --token string       authentication token (PAT)
```
