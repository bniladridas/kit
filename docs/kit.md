## kit

A lightweight developer CLI tool

### Synopsis

Kit is a lightweight, extensible CLI for developer workflows and API interactions.

### Options

```
      --config string   config file (default is $HOME/.kit/kit.yaml)
  -h, --help            help for kit
  -v, --verbose         verbose output
```
